This plugin transforms a Json Payload to a PDF via XHTML/CSS based on the provided Velocity templates. The transformed formats will then be stored as Payloads. The transformer utilises Flying saucer and IText to transform the XHTML into PDF.

This plugin is currently maintained by the Queensland Cyber Infrastructure Foundation.
